## 확인된 방향

- 앱 이름은 `Car Wikipedia` 방향으로 정리한다.
- 솔루션/네임스페이스/실행 파일명은 각각 `CarWikipedia.sln`, `CarWikipedia`, `CarWikipedia.exe`로 맞춘다.
- 1차 목표는 현재 `.docx`로 저장/기록한 차량 자료를 `.md`로 안전하게 변환하는 것이다.
- 2차 목표는 변환된 Markdown 자료를 WPF 앱에서 검색, 조회, 삽입, 삭제하기 쉽게 만드는 것이다.
- 차량 문서는 로컬 Markdown 파일을 원본 데이터로 사용한다.
- 화면은 읽기 우선 뷰를 기본으로 하고, 편집은 보조 모드로 제공한다.
- DOCX는 원본을 보존하고 Markdown으로 가져오는 import 대상이다.
- 주요 사용자는 장기렌트 상담 중 차량 가격, 트림, 주요 옵션, 상담 포인트를 빠르게 찾는 영업 사용자다.
- V1에서는 웹 크롤링, AI 요약, 자동 차량 비교, CRM/견적/계약 기능을 만들지 않는다.
